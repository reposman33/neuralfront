Fruit

This application applies image recognition to fruit. Upload an image of some fruit or hold a piece of fruit in front of the camera, and this application will identify the type of fruit.

Use the Camera

Photograph some fruit using the camera. Your browser will request access to the camera.

Upload an Image

Upload an existing image from your file system. It currently only works with .jpg files.
